# Build it yourself – Frontend Prototype

Ein schlanker React/Vite-Frontend-Prototyp für deine App-Idee.

## Start

```bash
npm install
npm run dev
```

Danach öffnet Vite lokal die Vorschau im Browser.

## Enthalten

- Landing Page mit Branding und Einstieg
- Modulauswahl nach Bauteilkategorien
- Erste vollständige Anleitung für `W4 – Holzständerwand EG`
- Platzhalter für weitere Module
- Reines Frontend, damit du Design und Inhalte direkt weiterentwickeln kannst

## Nächste sinnvolle Schritte

1. Echte Renderings oder Screenshots pro Step einbauen
2. Daten in eigene JSON-Dateien auslagern
3. Routing ergänzen (z. B. React Router)
4. Später als React-Native- oder Expo-App für iOS weiterdenken
